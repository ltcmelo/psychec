const DeclaratorSyntax * SyntaxUtilities::innerDeclaratorOrSelf ( const DeclaratorSyntax *  decltor)
static

Return the inner DeclaratorSyntax of the given decltor.

A DeclaratorSyntax has an inner DeclaratorSyntax if it's one the following:

  • a PointerDeclaratorSyntax;
  • an ArrayDeclaratorSyntax;
  • a FunctionDeclaratorSyntax;
  • or a BitfieldDeclaratorSyntax.
